# hellorook-public

Public issue tracker and feedback repository for [hellorook.ai](https://hellorook.ai).

## What is Rook?

Rook is an AI agent from Hypha Worker Co-operative focused on private, sovereign knowledge access. It is designed to search and synthesize your documents without handing your data to third-party cloud services.

The public site describes Rook as:

- sovereign intelligence for the solidarity economy and beyond
- private, secure LLM-powered knowledge management
- able to work in the browser, terminal, and chat
- designed to run on your machine or your server
